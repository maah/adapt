# Adapt (Discord Bot)

Adapt is a simple bot made for a small private server.  
Its goal is to bring everyone together, focused on personalization, conversions, and translation.

## Commands
- **Role color**  
/color get  
/color set

- **Currency conversion**  
/currency common  
/currency any  
/currency find

- **Unit conversion**  
/unit area  
/unit data  
/unit length  
/unit mass  
/unit speed  
/unit temperature  
/unit volume  
/unit time

- **Time zone conversion, display time**  
/time  
/time convert

- **Translation**  
/translate

- **\[Owner-only\] Birthday message updater**  
/birthday add  
/birthday edit  
/birthday remove
